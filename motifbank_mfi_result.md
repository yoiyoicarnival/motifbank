◆ 物理的解釈
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  MFI は 12 種の結晶学的に非等価な T サイト (Si 位置) を持つ。
  スーパーセルを拡大しても「新しい局所環境の種類」は増えない。
  → bank に貯めるべき fragment エネルギーは有限個 (663 型) で打ち止め。

  MotifBank の speedup = (系のサイズ) / (bank の型数)
                       = N / 663    ← N に比例して成長

◆ H-capped Si(OH)4 断片化 (2026-05-17)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  問題: SiO4 は [SiO4]^4- = charge=-4, PySCF charge=0 で SCF 未収束
  解決: dangling bond を H でキャップ → Si(OH)4 (中性, charge=0)

  実装:  mol_type = "si_oh4"
         Si + 4O (最近傍) + 4H (O-Si 延長上, O-H=0.96Å)
  検証:
    ✅ 9原子/断片  (Si + 4O + 4H)
    ✅ Si-O = 1.608~1.611 Å  (文献値 ~1.61 Å)
    ✅ O-H = 0.960 Å  (配置通り)
    ✅ PySCF HF/STO-3G charge=0 で収束
    ✅ Phase 0 が si_oh4 でも維持  (gamma=0.063, DEPLOY)

  sio4 vs si_oh4 bank サイズ比較:
    N_SiO4   bank(sio4)   bank(si_oh4)
    ─────────────────────────────────
      96          528          526  (≒同じ)
     384          577          572
     768          663          644
    1536          663          644  ← 飽和点は同じ

  結論: H-capping は Phase 0 を壊さない。
        bankサイズの変化 < 3% (geom_key が幾何形状のみを見るため)

◆ S_local: ローカル幾何エントロピー (新概念)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  定義: S_local(material) = log( N_bank_sat )
        N_bank_sat = bank が飽和したときのエントリ数

  MFI silicalite-1:
    N_bank_sat = 644 (si_oh4)  または  663 (sio4)
    S_local    = log(644) = 6.47 nats = 9.33 bits

  物理的意味:
    - S_local が有限 = 局所幾何の種類が有限 = 周期結晶の定義そのもの
    - S_local が小さい材料 = MotifBank の効果が大きい (Phase 0 に近い)
    - S_local = ∞ (= log N) のとき Phase 3 (ランダム系)

  材料による S_local の予測:
    材料                  S_local 予測    Phase
    ─────────────────────────────────────────
    MFI silicalite-1      6.5 nats        0  (実測)
    ice Ih                ~4 nats         0  (実測: N_bank=16)
    cristobalite          ~5 nats         0  (実測: N_bank=32)
    混合カチオンゼオライト ~8 nats         2  (LTA 実測)
    アモルファス           → ∞             3
